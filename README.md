# Ejercicio Práctico - Clases y Objetos en JavaScript

## Descripción
Este proyecto contiene ejemplos prácticos de modelado de entidades usando Programación Orientada a Objetos (POO) en JavaScript. Incluye la definición de clases, atributos y métodos, así como la creación de objetos y su uso.

## Archivos incluidos
- **ClaseAlumno.js**: Clase que modela a un alumno con atributos y método para mostrar su información.
- **ClaseBandaMusical.js**: Clase que representa una banda musical, con métodos para mostrar información y listar discos.
- **clasePerro.js**: Clase que modela a un perro, con métodos para mostrar sus atributos y simular un ladrido.

## Ejecución
Puedes ejecutar cada archivo por separado en Node.js para ver los ejemplos de uso y la salida en consola:

```bash
node ClaseAlumno.js
node ClaseBandaMusical.js
node clasePerro.js
```

## Objetivo
- Practicar la creación de clases y objetos en JavaScript.
- Modelar entidades de la vida real y simular sus comportamientos.
